海湾安全技术有限公司厕所文化交流博客

每周一更，欢迎投稿。

内容包括头脑风暴，时事分享，前沿科技，技术知识。



## 创立目的：

- 拉屎的时候，远离社交媒体，“用力”思考，保护“菊”部健康。
- 关注前沿的科技知识。
- 提高整体的技术水平。
- 引起人们对文化、科技、经济等方面的思考。
